# 💰 Employee Salary Prediction

A Machine Learning web application built with **Python** and **Streamlit** that predicts an employee's salary based on different factors such as age, work experience, city tier, performance score, skills, and remote work status.

---

## 📌 Features

- 🎯 Predict employee salary instantly
- 👨‍💼 User-friendly Streamlit interface
- 📊 Machine Learning based prediction
- 🏙️ City Tier selection
- ⭐ Performance Score input
- 💻 Skills and Experience based prediction
- 🌐 Remote Work support
- 🎨 Attractive UI with custom CSS

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Joblib
- Machine Learning

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Employee-Salary-Prediction.git
```

### Go to Project Folder

```bash
cd Employee-Salary-Prediction
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```

---

## 📊 Input Parameters

- Age
- Years of Experience
- City Tier
- Performance Score
- Number of Skills
- Remote Work (Yes/No)

---

## 🎯 Output

The application predicts the estimated employee salary using the trained Machine Learning model.
